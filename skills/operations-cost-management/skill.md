# Operations Cost Management Skill

- Source Spec: `docs/06_operations/cost_management.md`
- Upstream: `architecture-deployment`, `operations-iac`
- Downstream: `risk-management`

## Workflow
1. Estimate per-resource monthly cost from the deployment architecture.
2. Set budget thresholds and anomaly alerts with owners.
3. Define optimization levers (right-sizing, commitments, tiering, off-hours).
4. Enforce tagging policy for cost attribution.

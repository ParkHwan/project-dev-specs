# Suggested Commands

## File Operations
* `ls -R`: List files recursively.
* `grep -r "pattern" .`: Search for patterns in files.
* `cat <file>`: Read file content.

## Diagramming
* `mermaid-cli`: (If installed) Render Mermaid diagrams to images.
* Use [Mermaid Live Editor](https://mermaid.live/) for previewing diagrams.

## Markdown Tools
* Use a Markdown linter (e.g., `markdownlint`) if available to ensure document quality.
